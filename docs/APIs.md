### Upload_task
* Description: For uploading the new task form
* URL: `/api/tasks/new_task`
* Method: `POST`
* Request Example:
```javascript
{
 body:{% Gene_task_form %}
}
```
* Response Example on `success`:
```
http/1.1 200 OK
```
```
{
 success:true,
 data:{
  task_id:’1111…’
 }
}
```
* Response Example on `failure`:
```
http/1.1 200 OK
```
```javascript
{
 success:false,
 errCode:INFOMATION_UNCOMPLETED
}
```
